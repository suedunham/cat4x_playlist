## [0.1.0] - 2026-03-08

First release.

### Added

* Edited scripts from [CaTRoX_QWR](https://github.com/TheQwertiest/CaTRoX_QWR) 
  v5.0.0 for the common and playlist viewer packages. Several customizations 
  desired in my personal theme have probably been included but not recorded.
* WINE registry script for changing colors of application GUI objects as per the 
  older "(no theme)" setting.

### Fixed

* Crash in new WINEPREFIXes due to missing DPI registry key.
* Literal RGB color and font values made into panel properties. The default 
  values have been changed from those original colors and Segoe UI font faces.
* Scrollbar rendering improved/ruined, depending on one's perspective.
